# SQL examples
## Prérequis  
```
pip install Flask
pip install mysql-connector-python  
```

Add an envrionment variable in the console  
```
set MYSQL_FLASK_PWD=<redacted>>
```

Start server
```
cd 2-BasicSQL
flask run
```
Test: [Read SQL](http://127.0.0.1:5000/readSQL)  
Test avec Postman: voir collection Flask dans PostMan
